World Wide Web Consortium (W3C) created CSS.
A CSS selector selects the HTML element(s) you want to style.

CSS Selectors
---------------------
1)simple selector
   Select element based on id,class,name

2)Combination selector
    Applying styles to two or more tags at a time

3)Pseudo class selector
    Every HTML Attribute  has certian state.Apply styles based on the state

4)Pseudo element selector
    select and style a part of an element
    
5)Attribute selector
     select elements based on an attribute or attribute value
